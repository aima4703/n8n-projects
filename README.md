Social Media Mentions \& Sentiment Tracker



What the workflow does?

This workflow monitors mentions of a brand, topic, or keyword across social media platforms (RSS feeds).  

It extracts post content, author, timestamp, and URL, performs basic sentiment analysis (positive, neutral, negative), logs everything in Google Sheets, and sends email notifications for negative mentions or trending topics.



Tools Used

\- n8n (Workflow Automation)

\- RSS feeds

\- Google Sheets

\- Gmail

\- JavaScript(for basic sentiment analysis in Function nodes)
